# Spacing & Layout Tokens

Spacing tokens provide a consistent scale for margins, padding, gaps, and other layout properties throughout the design system.

## Border Radius

Border radius tokens control the roundness of corners for UI elements.

## Usage Examples

### Border Radius

```tsx
// Cards and containers
<div className="rounded-lg border p-4">
  Card with large radius
</div>

// Buttons
<button className="rounded-md px-4 py-2">
  Button with medium radius
</button>

// Pills and badges
<span className="rounded-full px-3 py-1">
  Badge
</span>

// Images
<img className="rounded-xl" src="..." alt="..." />
```

### Padding

```tsx
// Component padding
<div className="p-4">Small padding</div>
<div className="p-6">Medium padding</div>
<div className="p-8">Large padding</div>

// Directional padding
<div className="px-4 py-2">Horizontal and vertical</div>
<div className="pt-8 pb-4">Top and bottom</div>
```

### Margin

```tsx
// Spacing between elements
<div className="mb-4">Element with bottom margin</div>
<div className="mt-8">Element with top margin</div>

// Negative margins
<div className="-mt-4">Negative top margin</div>
```

### Gap

```tsx
// Flex gap
<div className="flex gap-4">
  <div>Item 1</div>
  <div>Item 2</div>
</div>

// Grid gap
<div className="grid grid-cols-3 gap-6">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

### Space Between

```tsx
// Vertical spacing
<div className="space-y-4">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>

// Horizontal spacing
<div className="flex space-x-2">
  <button>Button 1</button>
  <button>Button 2</button>
</div>
```

## CSS Variables

```css
/* Border radius */
.custom-card {
  border-radius: var(--radius-lg);
}

.custom-button {
  border-radius: var(--radius-md);
}

/* Using Tailwind spacing scale */
.custom-container {
  padding: theme("spacing.4"); /* 16px */
  margin-bottom: theme("spacing.6"); /* 24px */
}
```

## Best Practices

1. **Use consistent spacing**: Stick to the spacing scale for all layout decisions
2. **Prefer gap over margin**: Use `gap` in flex/grid layouts instead of margins when possible
3. **Use space utilities**: Use `space-x-*` and `space-y-*` for consistent spacing between children
4. **Match radius to size**: Larger elements can use larger border radius values
5. **Consider touch targets**: Ensure interactive elements have adequate padding (minimum 44x44px)
6. **Responsive spacing**: Adjust spacing for different breakpoints when needed

## Common Patterns

### Card Spacing

```tsx
<div className="space-y-4 rounded-lg border p-6">
  <h3 className="text-lg font-semibold">Card Title</h3>
  <p className="text-muted-foreground text-sm">Card content</p>
</div>
```

### Button Spacing

```tsx
<button className="rounded-md px-4 py-2">
  Button Text
</button>

<button className="rounded-md px-3 py-1.5 text-sm">
  Small Button
</button>
```

### Form Spacing

```tsx
<form className="space-y-6">
  <div className="space-y-2">
    <label>Label</label>
    <input className="rounded-md px-3 py-2" />
  </div>
  <button className="rounded-md px-4 py-2">Submit</button>
</form>
```
